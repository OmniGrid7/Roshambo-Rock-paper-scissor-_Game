# Roshambo (Rock, Paper, Scissors)

A clean, interactive implementation of the classic game **Roshambo**, built specifically for solo play against a computer opponent.

## 🕹️ Features
*   **Single-Player Mode**: Challenge a randomized AI logic in a fair, one-on-one match.
*   **Real-Time Scoreboard**: Automatically tracks your wins, losses, and ties throughout your session.
*   **Input Validation**: Built-in checks to ensure only valid moves (Rock, Paper, or Scissors) are accepted.
*   **Endless Play**: Play as many rounds as you like and quit whenever you're ready.

## 🚀 How to Play
1.  **Launch the Game**: Run the script in your terminal or browser.
2.  **Make Your Move**: Enter your choice: **Rock**, **Paper**, or **Scissors**.
3.  **The Computer's Turn**: The computer will simultaneously generate a random selection.
4.  **See Who Won**: The winner is determined by classic rules:
    *   **Rock** blunts **Scissors**
    *   **Scissors** cuts **Paper**
    *   **Paper** covers **Rock**

## 🛠️ Built With
*   **[JavaScript](https://github.com/topics/rock-paper-scissors)** (utilising the `random` module for computer choices)
*   **[Markdown](https://github.com)** (for documentation)
